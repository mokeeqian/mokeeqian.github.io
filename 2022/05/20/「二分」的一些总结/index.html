「二分」的一些总结

序言

二分其实还挺难的，边界搞不清楚的话，做题有时候稀里糊涂就提交AC了，至于对错，只能自求多福。
最近写二分越发觉得不会设置边界条件了，所以开一篇帖子学习一下。

二分

什么是二分，最多使用的可能就是在有序数组中查找某个目标值，使用二分查找，通过选取 midmidmid，不断比较 nums[mid]nums[mid]nums[mid]和targettargettarget，如果前者小于后者则在右半区间继续搜索；否则，在左半区间搜索。直至查找成功，或者失败（区间长度为0）。

那么，如果数组经过了某种变换后（比如旋转、循环移位），还能不能用二分？

能！

二分的本质是「二段性」，而不仅仅是「单调性」。只要一段满足某种性质，另外一段不满足某种性质，就能使用二分！

查找某个目标值

  1. 左闭右闭区间 [left,right][left, right][left,right]
    int bin_search(int[] nums) {
    // 注意，初始区间[left, right]，right能取到，所以是nums.length-1
    int left = 0, right = nums.length-1;
    // 注意是 <=
    while ( left <= right ) {
        int mid = left + right >> 1;
        if ( nums[mid] == target ) {
            return mid;
        } else if ( nums[mid] > target ) {
            right = mid-1;
        } else {
            left = mid+1;
        }
    }
}
  2. 左闭右开区间 [left,right)[left, right)[left,right)
    int bin_search(int[] nums) {
    // 注意，初始区间[left, right]，right不能取到，所以是nums.length
    int left = 0, right = nums.length;
    // 注意是 <
    while ( left < right ) {
        int mid = left + right >> 1;
        if ( nums[mid] == target ) {
            return mid;
        } else if ( nums[mid] > target ) {
            // 右开区间，right始终扫描不到（nums[mid]已经不合法了）
            right = mid;
        } else {
            left = mid+1;
        }
    }
}
  • 如果存在多个符合条件的元素，最终返回的是最中间的那个

浮点数二分

boolean check(double x) {/* ... */} // 检查x是否满足某种性质

double bsearch_3(double l, double r)
{
    const double eps = 1e-6;   // eps 表示精度，取决于题目对精度的要求
    while (r - l > eps)
    {
        double mid = (l + r) / 2;
        if (check(mid)) 
            r = mid;
        else l = mid;
    }
    return l;
}

上下界搜索

  • 如果数组不是严格单调的，即存在重复元素，要求问target第一次出现的位置和最后一次出现的位置？
  • 求大于target的最小的元素出现的位置？
  • 等等等

  1. 区间下界

    左闭右闭区间

    int left_bound(int[] nums, int target) {
    int left = 0, right = nums.length - 1;
    while (left <= right) {
        int mid = left + (right - left) / 2;
        if (nums[mid] < target) {
            left = mid + 1;
        } else if (nums[mid] > target) {
            right = mid - 1;
        } else if (nums[mid] == target) {
            // 别返回，锁定左侧边界
            right = mid - 1;
        }
    }
    // 最后要检查 left 越界的情况
    if (left >= nums.length || nums[left] != target)
        return -1;
    return left;
}

    左闭右开区间

    int left_bound(int[] nums, int target) {
    if (nums.length == 0) return -1;
    int left = 0;
    int right = nums.length; // 注意 此时为开区间

    while (left < right) { // 注意  开区间则为 < 
        int mid = left + (right - left) / 2;
        if (nums[mid] == target) {
            right = mid;
        } else if (nums[mid] < target) {
            left = mid + 1;
        } else if (nums[mid] > target) {
            right = mid; // 注意
        }
    }
    // 如果不存在，left是target应该被插入的位置
    return left;
}

  2. 区间上界

    左闭右闭区间

    int right_bound(int[] nums, int target) {
    int left = 0, right = nums.length - 1;
    while (left <= right) {
        int mid = left + (right - left) / 2;
        if (nums[mid] < target) {
            left = mid + 1;
        } else if (nums[mid] > target) {
            right = mid - 1;
        } else if (nums[mid] == target) {
            // 别返回，锁定右侧边界
            left = mid + 1;
        }
    }
    // 最后要检查 right 越界的情况
    if (right < 0 || nums[right] != target)
        return -1;
    return right;
}

    左闭右开区间

    int right_bound(int[] nums, int target) {
    if (nums.length == 0) return -1;
    int left = 0, right = nums.length;

    while (left < right) {
        int mid = left + (right - left) / 2;
        if (nums[mid] == target) {
            left = mid + 1; // 注意
        } else if (nums[mid] < target) {
            left = mid + 1;
        } else if (nums[mid] > target) {
            right = mid;
        }
    }
    return left - 1; // 注意
}
